Module 2 Grad Cafe Web Scraper

This project builds a Python web scraping pipeline for publicly available Grad Cafe applicant data. It checks robots.txt, constructs URLs with urllib, parses rendered applicant listings with BeautifulSoup and regex, and stores structured applicant records as JSON.

The scraper is designed to collect 50,000 applicant records, preserve raw listing text for traceability, and clean program and university names using the local LLM standardization package provided with the assignment.
