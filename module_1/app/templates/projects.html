Module 1 Flask Portfolio Website

Module 2 Grad Cafe Web Scraper

This project builds a Python web scraping pipeline for publicly available Grad Cafe applicant data. It checks robots.txt, constructs URLs with urllib, parses rendered applicant listings with BeautifulSoup and regex, and stores structured applicant records as JSON.

The scraper is designed to collect 50,000 applicant records, preserve raw listing text for traceability, and clean program and university names using the local LLM standardization package provided with the assignment.

View Module 2 GitHub Folder

Module 3 Grad Cafe Database Queries and Analysis

This project loads cleaned Grad Cafe applicant records from Module 2 into a PostgreSQL database, then uses SQL queries to analyze application terms, applicant origin, admissions outcomes, GPA and GRE metrics, degree type, universities, and program fields.

The module also includes a styled Flask analysis page that displays query results dynamically from the database, plus Pull Data and Update Analysis controls for refreshing the dataset and webpage results.
